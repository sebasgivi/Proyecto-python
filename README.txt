Para ejecutar el juego "Meteoritos vs Nave" abra el archivo Main.py 
el cual es la clase encargada de ejecutar el juego, corralo normalmente
segun su aplicacion para programar y comience a divertirse, seleccione su idioma
y juegue con "w", "a"y "d" como en un juego comun pero con el añadido de que toca
presionar "enter" para enviar la orden y ademas utiliza F y P para disparar y pausar
respectivamente el objetivo del juego es conseguir el puntaje mas alto, teniendo en 
cuenta que las compras en la tienda sacrifican tu puntaje.
